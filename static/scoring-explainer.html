節税・団信型ペルソナは意図的に除外

業界で「カモ化」批判がある営業手法（新築ワンルーム節税）はペルソナ化せず、L-008 では推奨対象から除外します。 てるさんの哲学（営業利益率向上＝健全な投資判断）と倫理整合させるための明示的な選択です。

データソースとカバレッジ

3サイトの強みを組み合わせて4,171件をカバー。Sランクは実利回り掲載の信頼度を担保するため HOME'S 限定としています。

なぜ S ランクは HOME'S 限定なのか？

SUUMO/ふれんずは利回り未掲載のため、エリア中央値からの推計利回りを使います。 推計値の信頼度を考慮し、collection_score の収益軸は半分（最大15点）にクリップ。 結果として実利回り掲載物件のみが S ランクの 80点以上に到達できる設計です。 SUUMO/ふれんずの優良物件は A ランクとペルソナマッチで救済しています。

スコープ外決定（透明性のために）

技術的に取れるサイトでも、規約・倫理・LPM のレピュテーションを優先して除外しているサイトがあります。

楽待
EXCLUDED
理由

利用規約 第10条 第1項(6)号で「クローラー・RPA等のスクレイピング」を名指しで禁止。LPM 法人としてのレピュテーションリスクが許容不可。

代替

入社後 LPM 経由で公式パートナー API の問い合わせを検討

健美家
EXCLUDED
理由

robots.txt で ClaudeBot 等を名指し disallow + 利用規約が全URL 404 で非公開 + 2024/9 に20万件情報漏洩事故あり警戒度高。

代替

入社後 LPM 経由で公式パートナー API の問い合わせを検討

レインズ
DEFERRED
理由

不動産流通機構の指定流通機構データベース。宅地建物取引業者ライセンス必須。API公開なし、個人ではアクセス不可。

代替

2026-04-20 LPM 入社後、宅建業者ステータス確認のうえ正式接続を検討
